# Antigravity 核心功能與入門指南簡報

這是一份專為初學者設計的 **Antigravity** (Google DeepMind 開發的代理型 AI 程式設計助手) 入門簡報。

## 專案結構
- `index.html` - 簡報主頁面 (內建簡報播放器)
- `style.css` - 簡報樣式表 (支援深色/淺色模式切換與精美動畫)
- `slides.js` - 簡報資料庫 (包含 52 頁精美投影片內容)
- `app.js` - 簡報播放器邏輯控制 (鍵盤操控、目錄搜尋、大綱檢視)

## 如何閱讀簡報
1. 直接在瀏覽器中雙擊打開 `index.html` 即可開始播放。
2. 支援以下控制方式：
   - **向右方向鍵 (→) / 空白鍵 (Space) / 下一頁按鈕**：切換到下一張投影片。
   - **向左方向鍵 (←) / 上一頁按鈕**：切換到上一張投影片。
   - **主題切換按鈕**：可在深色模式 (預設) 與淺色模式之間進行切換。
   - **目錄大綱按鈕 (Outline)**：開啟側邊欄檢視所有 52 張投影片，並可透過關鍵字搜尋快速跳轉。
   - **簡報模式切換 (Fullscreen)**：將投影片放大至全螢幕。
